Unplugin Turbo Console is a universal build plugin that transforms the way you use `console` in frontend development. It adds color-coded output, clickable links that open your editor at the exact source line, bidirectional log passing between server and client, and a real-time console inspector — all with zero changes to your existing code.

## Why Unplugin Turbo Console?

Standard `console.log` output gives you a value and a file name — that's it. When you're debugging across multiple files in a large codebase, you spend more time scanning output than fixing bugs. Unplugin Turbo Console gives each log statement a rich identity: the framework file type, the variable name, the exact source line, and a clickable link back to your editor.

<Steps>
  <Step title="Install the package">
    Add `unplugin-turbo-console` as a dev dependency with your package manager of choice.
  </Step>
  <Step title="Add the plugin">
    Register the plugin in your build tool config — Vite, Webpack, Nuxt, SvelteKit, Astro, and more are all supported.
  </Step>
  <Step title="Write console statements as normal">
    No code changes required. Every existing `console.log`, `console.warn`, and `console.error` is automatically enhanced.
  </Step>
  <Step title="Click to open your editor">
    Each log line in the browser includes a link. Click it to jump directly to the source line.
  </Step>
</Steps>
